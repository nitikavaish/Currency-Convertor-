# 💱 Currency Converter

A simple and responsive Currency Converter web application built using HTML, CSS, and JavaScript. It fetches real-time exchange rates through an API, allowing users to convert currencies quickly and accurately.

## Features
- Real-time currency conversion
- Multiple currency support
- Responsive design
- Clean and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Exchange Rate API
